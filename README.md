# Gaming and Chill

Gaming and Chill is my portfolio project and probably one of the most important projects I will ever have.
The concept of the app is fairly simple.
You are able to search games as a user, preview their trailer, categorize them with an advanced filter,
have your own profile, save the games you like in your own library, read patch notes and news based on games
and be able to download them so you can start playing!!!!

### Part of me!!

The reason of this project been the most important to me is because of it's scalability!
Adding more and more features teaches me about concepts,ideas and technologies that I haven't practiced or worked so far!
etc( making my application mobile friendly allowed me to work with media queries, creating my own fuzzy search or breaking parts of a component to make them individual reusable components like a button for example!!! (CardButton component _ I really love this one _) )!
So in the end I can't stop thinking about the endless posibilities and features I can implement!!!
